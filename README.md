## About this project
โปรเจกต์เว็บไซต์ Lost&Found จัดทำขึ้นเพื่อพัฒนาเว็บไซต์ที่ช่วยอำนวยความสะดวกให้กับนักศึกษาและบุคลากรภายในสถาบันในการแจ้งของสูญหายและของที่พบ โดยมีวัตถุประสงค์เพื่อให้สามารถประกาศ แจ้ง และค้นหาข้อมูลของสูญหายได้อย่างรวดเร็ว มีประสิทธิภาพ และลดปัญหาความยุ่งยากจากการประกาศผ่านช่องทางอื่น ๆ เช่น Facebook Instagram ของช่องทางหลัก KMITL หรือบอร์ดประชาสัมพันธ์
